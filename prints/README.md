# 3D Print Files

Hardware attachments designed in Fusion 360 / Onshape for the Cura feeding assistant.

## Parts

### bottle-cradle.stl
Fixed cradle that holds the water bottle in a known position on the table. The arm always reaches to this exact location — eliminates the need for vision-guided grasping in V1. Print in PLA, 20% infill.

### utensil-holder.stl
Rack with fixed slots for fork, spoon, and napkin. Each slot is at a known position so the arm can pick utensils without vision guidance (V3). Print in PLA, 20% infill.

### t5ai-stand.stl
Angled patient-facing stand for the T5AI board. Positions the screen and camera at face height for a seated patient. Adjustable angle 15°–45°. Print in PLA, 30% infill.

### gripper-adapter.stl (optional)
Custom gripper attachment for improved utensil grip (V3). Replaces the default 2-finger gripper tips.

## Print Settings
- Material: PLA
- Layer height: 0.2mm
- Infill: 20% (structural parts 30%)
- Supports: As needed
- Scale: 100% (designed to exact dimensions)
